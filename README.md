# PHP Bug Bounty / Vulnerability Reporting System

## Description
A PHP-based bug bounty platform where users can submit vulnerabilities. Admin can approve/reject reports.  

## Features (Implemented)
- User submission form with file upload
- Prepared statements for secure DB inserts
- Password hashing for admin
- Display approved reports
- Input/output sanitization
